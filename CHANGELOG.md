### 新增

- 新增入门引导流程，首次打开应用时会说明通知监听、关键词、跳过按钮文本、休息日规则和执行记录的使用方式。
- 主页面新增“使用引导”入口，用户可以随时重新查看引导说明。
- 新增“休息日规则”配置，支持节假日 + 双休、节假日 + 单休、节假日 + 大小周、周期排班和完全自定义。
- 新增完全自定义休息日模式，可在应用内标记未来 30 天的休息日期。
- 新增自定义云端节假日数据地址，支持使用自己的 `holiday.json` 数据源。
- 新增自建节假日数据指南，说明 JSON 格式、字段含义和数据维护方式。
- 设置页新增“节假日数据”分区，可配置云端数据地址并打开自建数据指南。
- 调休日历更新会使用当前配置的数据地址，保存自定义地址前会先检测并缓存一份可用数据。

### 修复

- 修复休息日判断只能依赖节假日数据的问题，现在会按当前休息日规则判断今天是否休息。

### 优化

- 主页面增加当前休息日规则摘要，方便确认应用正在按哪种规则判断今天是否休息。
- 隐私政策和 README 已同步更新，补充休息日规则、自定义数据地址和本地数据保存说明。
